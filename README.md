# Task 1 - Linear Regression Model for House Price Prediction

## Overview

This project was completed as part of my AI & Machine Learning Internship.

The objective of this task is to build a Linear Regression model that predicts house prices using the California Housing dataset. The project follows the basic machine learning workflow, including data exploration, visualization, model training, prediction, evaluation, and saving the trained model.

---

## Dataset

- **Dataset:** California Housing Dataset
- **Source:** Scikit-learn
- **Type:** Regression Dataset

The dataset contains information about different housing districts in California, including features such as median income, average number of rooms, population, latitude, longitude, and median house value.

---

## Project Workflow

The following steps were completed during this project:

1. Imported the required libraries.
2. Loaded the California Housing dataset.
3. Explored the dataset using statistical analysis.
4. Checked for missing and duplicate values.
5. Visualized the dataset using different plots.
6. Prepared the data for model training.
7. Split the dataset into training and testing sets.
8. Trained a Linear Regression model.
9. Predicted house prices using the trained model.
10. Evaluated the model using MAE, RMSE, and R² Score.
11. Saved the trained model using Joblib.
